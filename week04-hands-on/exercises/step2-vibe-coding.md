# Step 2: 바이브코딩 체험 (30분)

> 방금 만든 CLAUDE.md를 활용해서, AI에게 코드를 시켜봅니다.
> "바이브코딩" = 한국어로 설명하면 AI가 코드를 만들어주는 것

---

## 바이브코딩이란?

```
내가 할 일:  한국어로 "이거 해줘" 설명
AI가 할 일:  Python 코드를 만들어줌
내가 할 일:  실행 버튼만 누르기
```

코드를 외울 필요 없습니다. **한국어로 잘 설명하는 것**이 핵심입니다.

---

## 실습 A: CLAUDE.md 없이 시켜보기 (10분)

먼저, CLAUDE.md **없이** AI에게 시켜봅시다.

### 방법 1: Claude.ai에서 하기
1. [claude.ai](https://claude.ai) 접속
2. 아래 프롬프트를 **그대로** 복사해서 붙여넣기:

```
경비내역 엑셀 파일을 정리하는 Python 코드 만들어줘.
```

3. AI가 만들어준 결과를 보세요 — 아마 우리 상황과 안 맞을 겁니다

### 방법 2: VSCode에서 Claude Code로 하기
1. VSCode에서 터미널 열기: `Ctrl + ~` (백틱)
2. `claude` 입력하고 Enter
3. 같은 프롬프트 입력:

```
경비내역 엑셀 파일을 정리하는 Python 코드 만들어줘.
```

→ **결과를 메모장에 복사해두세요** (나중에 비교할 겁니다)

---

## 실습 B: CLAUDE.md와 함께 시켜보기 (15분)

이번에는 Step 1에서 만든 CLAUDE.md의 내용을 **함께** 알려주고 시켜봅니다.

### 방법 1: Claude.ai에서 하기

**본인이 만든 CLAUDE.md 내용**을 프롬프트에 포함해서 보내세요.
아래는 예시입니다 (본인 CLAUDE.md 내용으로 바꿔서 사용):

```
아래는 내 프로젝트 설명이야:

---
# 경비 정리 자동화

## 업무 개요
매월 각 부서에서 올라온 경비 내역을 정리해서 보고서를 만든다.

## 입력 데이터
- 경비내역.xlsx: 일자, 부서, 항목, 금액, 결제방법이 들어있는 엑셀 파일

## 원하는 결과물
- 부서별 경비 합계표
- 항목별(교통비, 식대, 접대비 등) 합계
- 개인카드 사용 건수와 금액 따로 표시

## 기술 환경
- Python 3.12
- 한국어 주석 필수
- 결과물은 엑셀(.xlsx)로 저장
---

이 내용을 바탕으로 경비내역.xlsx를 정리하는 Python 코드를 만들어줘.
data/ 폴더에 경비내역.xlsx가 있어.
```

### 방법 2: VSCode에서 Claude Code로 하기

Claude Code는 **CLAUDE.md를 자동으로 읽어줍니다!**
같은 폴더에 CLAUDE.md가 있으면, 간단하게만 시키면 됩니다:

```
data/경비내역.xlsx를 읽어서 CLAUDE.md에 적은 대로 정리해줘.
부서별 합계, 항목별 합계, 개인카드 건수를 포함해서
결과를 data/경비정리결과.xlsx로 저장해줘.
```

---

## 결과 비교하기 (5분)

| 비교 항목 | CLAUDE.md 없이 | CLAUDE.md 포함 |
|----------|---------------|---------------|
| 컬럼명이 맞는가? | △ 추측해서 만듦 | ○ 정확하게 맞춤 |
| 부서별 합계가 있는가? | △ 있을 수도 없을 수도 | ○ 요청대로 포함 |
| 개인카드 따로 표시? | × 아마 없음 | ○ 요청대로 포함 |
| 한국어 주석? | △ 영어일 수도 | ○ 한국어로 작성 |
| 파일 저장 형식? | △ csv일 수도 | ○ .xlsx로 저장 |

**결론: CLAUDE.md(= 구체적인 설명)가 있으면 AI의 정확도가 올라간다!**

---

## 만약 코드가 에러가 난다면?

걱정하지 마세요. 에러 메시지를 AI에게 보여주면 됩니다.

```
이 코드를 실행했더니 이런 에러가 나:

[에러 메시지 복사 붙여넣기]

고쳐줘.
```

이것도 바이브코딩의 일부입니다. **에러가 나면 → AI한테 고쳐달라고 하기.**
