## 방법 A: 이미 정리된 문서가 있는 경우 (대부분 이쪽)

업무 관련 문서(docx, pptx, 엑셀 양식 등)가 이미 있으면 **그 파일을 그대로 가져오세요.**

- 업무 매뉴얼, 인수인계서, 업무분장표, 과제 기술서 등 뭐든 OK
- 여러 개 있으면 가장 핵심적인 것 1~2개만
- **보안 주의**: 민감한 숫자/이름이 있으면 지우거나 가짜로 바꿔오세요

모여서 이 문서를 AI에게 주고 "CLAUDE.md로 바꿔줘"라고 시킬 겁니다.

```
기존 문서(docx 등)  →  AI에게 "CLAUDE.md로 정리해줘"  →  바이브코딩 시작
```

### 방법 B: 정리된 문서가 없는 경우

아래 질문에 메모장으로 간단히 적어오세요:

```
1. 내가 하는 업무는? (한 줄)
2. 입력은 뭔가? (어떤 파일/데이터, 어디서 오는지)
3. 출력은 뭔가? (최종 결과물)
4. 이 작업에서 가장 귀찮은 부분은?
5. 얼마나 자주 하는 업무인가?
```
