# Step 3: 여러 파일 합치기 체험 (20분)

> 법인별로 따로 있는 엑셀 파일 5개를 하나로 합쳐봅니다.
> 실제 업무에서 "법인 20개 파일 합치기"를 자동화하는 첫걸음입니다.

---

## 현실 문제

> "해외법인이 20개인데, 매달 각각 엑셀 파일을 보내옴.
> 하나하나 열어서 복사-붙여넣기 하는 데 반나절..."

오늘은 5개 법인 파일을 AI에게 시켜서 한번에 합쳐봅니다.

---

## data/ 폴더에 있는 파일 확인

| 파일명 | 내용 |
|--------|------|
| 법인_US01_미국.xlsx | 미국법인 1~3월 매출 (USD) |
| 법인_JP01_일본.xlsx | 일본법인 1~3월 매출 (JPY) |
| 법인_CN01_중국.xlsx | 중국법인 1~3월 매출 (CNY) |
| 법인_DE01_독일.xlsx | 독일법인 1~3월 매출 (EUR) |
| 법인_VN01_베트남.xlsx | 베트남법인 1~3월 매출 (VND) |

모두 같은 양식: 월, 계정과목, 통화, 금액, 비고

---

## AI에게 시켜보기

### Claude.ai에서 하는 경우

아래 프롬프트를 복사해서 Claude.ai에 붙여넣기:

```
Python 코드를 만들어줘.

상황:
- data/ 폴더에 "법인_"으로 시작하는 엑셀 파일이 5개 있어
- 파일명 예시: 법인_US01_미국.xlsx, 법인_JP01_일본.xlsx
- 각 파일의 컬럼: 월, 계정과목, 통화, 금액, 비고

해줄 일:
1) data/ 폴더에서 "법인_"으로 시작하는 .xlsx 파일을 모두 읽어
2) 각 파일에 "법인코드"와 "법인명" 컬럼을 추가해 (파일명에서 추출)
   - 법인_US01_미국.xlsx → 법인코드: US01, 법인명: 미국
3) 전부 하나로 합쳐
4) 법인별/월별 금액 합계를 피벗테이블처럼 정리해
5) 결과를 data/법인매출통합.xlsx로 저장해

조건:
- openpyxl 라이브러리 사용
- 한국어 주석 필수
- 초보자가 읽을 수 있게 간단하게
```

### VSCode Claude Code에서 하는 경우

CLAUDE.md가 있으면 더 간단하게:

```
data/ 폴더에 법인별 매출 엑셀 파일이 5개 있어.
"법인_"으로 시작하는 파일을 전부 읽어서 하나로 합치고,
파일명에서 법인코드와 법인명을 추출해서 컬럼으로 추가해줘.
법인별/월별 합계도 만들어서 data/법인매출통합.xlsx로 저장해줘.
```

---

## 코드를 받았으면

1. AI가 만들어준 코드를 VSCode에서 새 파일로 저장
   - 파일 → 새 파일 → `merge_sales.py` 로 저장
2. 터미널에서 실행:
   ```
   python merge_sales.py
   ```
3. `data/법인매출통합.xlsx` 파일이 생겼는지 확인
4. 엑셀로 열어서 결과 확인

---

## 에러가 나면?

에러 메시지를 그대로 AI에게 보여주세요:

```
이 코드를 실행했더니 에러가 나:

[에러 메시지 복사 붙여넣기]

고쳐줘.
```

**흔한 에러들:**
| 에러 | 원인 | 해결 |
|------|------|------|
| `ModuleNotFoundError: No module named 'openpyxl'` | openpyxl이 설치 안 됨 | 터미널에서 `python -m pip install openpyxl` 실행 |
| `FileNotFoundError` | 파일 경로가 다름 | data/ 폴더 위치 확인 |
| `PermissionError` | 엑셀 파일이 열려있음 | 엑셀 파일 닫고 다시 실행 |
