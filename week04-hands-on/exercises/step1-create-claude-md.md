# Step 1: CLAUDE.md 만들기 (25분)

> AI에게 "내가 누군지, 무슨 업무를 하는지" 알려주는 파일을 만들어봅니다.

---

## CLAUDE.md가 뭔가요?

신입사원이 들어오면 인수인계서를 주죠?
**CLAUDE.md는 AI에게 주는 인수인계서**입니다.

| 사람에게 | AI에게 |
|---------|--------|
| "우리 팀은 재경팀이야" | CLAUDE.md에 팀 소개 작성 |
| "매월 말에 환율 보고서 만들어" | CLAUDE.md에 업무 설명 작성 |
| "엑셀 양식은 이거야" | CLAUDE.md에 입출력 설명 작성 |

**CLAUDE.md가 있으면**: AI가 우리 상황을 알고 맞춤형 코드를 만들어줌
**CLAUDE.md가 없으면**: AI가 추측해서 엉뚱한 코드를 만들 수 있음

---

## 같이 만들어봅시다

### 1단계: 파일 만들기

1. VSCode에서 이 폴더(`week04-hands-on`)를 열었는지 확인
2. VSCode 왼쪽 탐색기에서 빈 공간 우클릭 → **새 파일**
3. 파일 이름: `CLAUDE.md`
4. Enter 누르면 빈 파일이 열립니다

### 2단계: 기존 문서로 CLAUDE.md 만들기

#### 방법 A: 기존 문서(docx 등)가 있는 경우 — 추천!

업무 매뉴얼, 인수인계서, 과제 기술서 등이 이미 있으면
**AI에게 변환을 시키면 됩니다.**

1. Claude.ai에 접속
2. 기존 문서 파일을 첨부 (docx, pptx, pdf 등 가능)
   - **보안 주의**: 민감한 숫자/이름이 있으면 미리 지우거나 가짜로 바꿔주세요
3. 아래 프롬프트를 붙여넣기:

```
첨부한 문서는 내 업무에 대한 설명이야.
이 내용을 아래 양식의 CLAUDE.md로 변환해줘:

# [업무명]

## 업무 개요
[한 문단으로 정리]

## 입력 데이터
- [어떤 파일/데이터가 필요한지]

## 원하는 결과물
- [최종 결과물이 뭔지]

## 기술 환경
- Python 3.12
- 한국어 주석 필수
- 결과물은 엑셀(.xlsx)로 저장
```

4. AI가 만들어준 결과를 VSCode의 CLAUDE.md에 복사-붙여넣기
5. 내용을 읽어보고 틀린 부분만 수정

#### 방법 B: 기존 문서가 없는 경우

아래 내용을 **한 줄씩 따라 치면서** 본인 업무에 맞게 바꿔보세요.

```markdown
# 경비 정리 자동화

## 업무 개요
매월 각 부서에서 올라온 경비 내역을 정리해서 보고서를 만든다.

## 입력 데이터
- 경비내역.xlsx: 일자, 부서, 항목, 금액, 결제방법이 들어있는 엑셀 파일

## 원하는 결과물
- 부서별 경비 합계표
- 항목별(교통비, 식대, 접대비 등) 합계
- 개인카드 사용 건수와 금액 따로 표시

## 기술 환경
- Python 3.12
- 한국어 주석 필수
- 결과물은 엑셀(.xlsx)로 저장
```

### 3단계: 본인 업무로 확인 & 보완

만들어진 CLAUDE.md를 읽어보고, **빠진 내용이 없는지** 확인하세요.

**영업관리팀이라면:**
```markdown
# 영업 실적 정리 자동화

## 업무 개요
매월 각 법인에서 올라온 매출 데이터를 통합 정리한다.

## 입력 데이터
- 법인별 매출 엑셀 파일 (법인_US01_미국.xlsx 등)
- 각 파일에 월, 계정과목, 통화, 금액, 비고 컬럼이 있음

## 원하는 결과물
- 전 법인 매출 통합표 (하나의 엑셀)
- 법인별/월별 매출 합계
- 전월 대비 증감률

## 기술 환경
- Python 3.12
- 한국어 주석 필수
- 결과물은 엑셀(.xlsx)로 저장
```

**MySQL + Flask 하시는 분이라면:**
```markdown
# 데이터 조회 웹 페이지

## 업무 개요
MySQL DB에 있는 데이터를 웹 화면에서 조회할 수 있게 만든다.

## 기술 환경
- Python 3.12, Flask
- MySQL (DBeaver로 확인 중)
- HTML 템플릿으로 화면 구성

## DB 정보
- 호스트: localhost
- DB명: (본인 DB명)
- 주요 테이블: (본인 테이블명)

## 원하는 결과물
- 웹 브라우저에서 데이터를 테이블로 보여주는 페이지
- 검색/필터 기능
```
