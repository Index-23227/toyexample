```

```

- AI가 `corporations`와 `monthly_sales`를 자동으로 JOIN함
- 법인명이 가나다순으로 정렬됨
- SQL을 한 글자도 안 썼는데 결과가 나옴

## 에러가 나면?

| 에러 | 원인 | 해결 |
|------|------|------|
| `OperationalError: no such table` | DB가 안 만들어짐 | `python data/create_db.py` 실행 |
| `no such column` | 컬럼명 오타 | CLAUDE.md의 스키마 확인 후 AI에게 알려주기 |
| `database is locked` | 다른 프로그램이 DB 사용 중 | DB Browser 등 닫고 다시 실행 |

에러 메시지를 그대로 AI에게 보여주면 고쳐줍니다:

```
이 에러가 나: [에러 메시지 붙여넣기]
고쳐줘.
```
