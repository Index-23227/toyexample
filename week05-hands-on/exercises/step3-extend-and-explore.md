## 기능 메뉴 — 골라서 시키기

아래 중 마음에 드는 것을 골라 AI에게 시키세요.
**쉬운 것부터 시작**하는 걸 추천합니다.

---

### A. 차트 추가 (추천! 제일 임팩트 큼)

```
메인 페이지에 법인별 매출 합계를 막대 차트로 보여줘.
Chart.js를 CDN으로 불러와서 사용하고,
표 위에 차트를 배치해줘.
원화 환산 금액 기준으로 보여줘.
```

> Chart.js = 설치 없이 바로 쓸 수 있는 차트 라이브러리

---

### B. 스타일 개선

```
대시보드를 더 보기 좋게 꾸며줘.
- 헤더에 회사 로고 영역 (텍스트로 "재경팀 매출 대시보드")
- 표에 호버 효과 (마우스 올리면 행 색 바뀜)
- 금액 열은 오른쪽 정렬
- 전체 폰트를 깔끔하게
- 반응형 (모바일에서도 보이게)
```

---

### C. 월별 필터

```
메인 페이지에 월 선택 드롭다운을 추가해줘.
"전체", "2026-01", "2026-02", "2026-03" 중에서 선택하면
해당 월의 매출만 표시되게 해줘.
```

---

### D. 환율 환산 토글

```
표에 "현지통화 / 원화" 전환 버튼을 추가해줘.
버튼을 누르면 금액이 현지통화 ↔ 원화로 바뀌게 해줘.
```

---

### E. 데이터 추가 폼 (도전!)

```
새로운 매출 데이터를 입력할 수 있는 페이지를 만들어줘.
URL: /add
- 법인 선택 (드롭다운)
- 월 입력
- 금액 입력
- 비고 입력
입력하면 DB에 INSERT하고, 메인 페이지로 이동해줘.
```

> 이걸 하면 **웹에서 데이터를 추가**할 수 있게 됩니다!
> 엑셀 파일을 열지 않아도 됩니다.

---

### F. CSV 다운로드

```
메인 페이지에 "CSV 다운로드" 버튼을 추가해줘.
클릭하면 법인별 매출 합계를 CSV 파일로 다운로드하게 해줘.
```

## 자유 실습: 내가 보고 싶은 화면 (20분)

위 메뉴를 다 해봤거나, 다른 걸 해보고 싶다면:

**"이런 화면이 있으면 좋겠는데"를 자유롭게 AI에게 시켜보세요.**

예시:
```
법인별 월별 매출을 히트맵 형태로 보여줘.
매출이 높으면 진한 색, 낮으면 연한 색.
```

```
전월 대비 증감률을 계산해서 화살표로 보여줘.
증가면 초록 ↑, 감소면 빨강 ↓
```

```
법인별 매출 비중을 파이 차트로 보여줘.
```

## 작게 시작해서 키우기 — 5단계

이번 실습 전체에 적용되는 원칙입니다:

```
1단계: 가장 단순한 버전을 먼저 만들기     ← Step 2에서 완료
2단계: 실제 데이터로 확인하기              ← Step 2에서 완료
3단계: 기능을 하나씩 추가하기              ← 지금 여기!
4단계: 에러 나면 AI에게 고치게 하기         ← 에러 = 정상. 당황 X
5단계: 모양 다듬기                        ← 마지막에 CSS/디자인
```

> **한 번에 다 시키지 마세요.** 하나 추가 → 확인 → 다음 하나.
> 한 번에 5가지를 시키면 AI도 헷갈리고, 에러가 나면 원인 찾기 어렵습니다.

---

## 에러 대응 팁

에러가 나면 AI에게 이렇게 보고하면 효과적입니다:

```
[상황] 차트를 추가하라고 했더니
[에러] 브라우저에서 이렇게 나와: (스크린샷 또는 에러 메시지)
[기대] 법인별 막대 차트가 표 위에 나와야 하는데
[실제] 차트가 안 보이고 빈 공간만 있어
고쳐줘.
```

이 **상황-에러-기대-실제** 패턴을 쓰면 AI가 더 정확하게 고칩니다.

---

## 오늘 배운 것 정리

1. **DB = 데이터 창고** — 엑셀 파일 대신 DB에 넣으면 SQL 한 줄로 원하는 데이터를 꺼낼 수 있다
2. **SQL은 AI가 써줌** — CLAUDE.md에 테이블 구조만 적어두면 AI가 정확한 SQL을 만든다
3. **Flask = 웹 페이지 만드는 도구** — `python app.py` 한 줄로 웹 서버가 뜨고, 브라우저에서 볼 수 있다
4. **작게 시작해서 키우기** — 기본 표 → 상세 페이지 → 차트 → 필터... 하나씩 추가
5. **에러 = 당연한 것** — AI에게 에러 메시지를 보여주면 고쳐줌
6. 이 모든 코드를 **AI가 만들어줬다** — 직접 SQL이나 Flask를 쓸 필요 없음
