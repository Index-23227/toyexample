## Flask 앱 요구사항

### 공통 규칙
- **라이브러리**: `flask`, `sqlite3` (Python 내장)
- **DB 경로**: `data/sales.db`
- **금액 표시**: 천 단위 콤마 (예: `125,000`)
- **한국어 UI**: 메뉴, 제목, 컬럼명 모두 한국어
- **정렬**: 법인명 가나다순 (독일 → 미국 → 베트남 → 일본 → 중국)

### 페이지 구성

#### 메인 페이지 (`/`)
법인별 매출 합계 표 + 원화 환산

| 법인명 | 통화 | 외화 합계 | 환율 | 원화 환산 |
|--------|------|-----------|------|-----------|
| 독일법인 | EUR | 304,000 | 1,480.0 | 449,920,000 |
| 미국법인 | USD | 375,000 | 1,350.0 | 506,250,000 |
| ... | | | | |

- 법인명 클릭 → 상세 페이지로 이동
- 원화 환산 = `외화 합계 × 환율` → 반올림 정수

#### 법인 상세 페이지 (`/corp/<corp_code>`)
해당 법인의 월별 매출 내역

| 월 | 금액 | 비고 |
|----|------|------|
| 2026-01 | 125,000 | |
| 2026-02 | 132,000 | 신규 거래처 |
| 2026-03 | 118,000 | |
| **합계** | **375,000** | |

- 월 오름차순 정렬
- 마지막에 합계 행

## 확장 기능 (Step 3에서 선택)

아래 중 원하는 것을 골라 AI에게 시킨다:

1. **차트 추가**: Chart.js CDN으로 법인별 매출 막대 차트
2. **월별 필터**: 드롭다운으로 특정 월만 보기
3. **환율 환산 토글**: 현지통화 / 원화 전환 버튼
4. **스타일 개선**: 표에 색상, 호버 효과, 반응형 레이아웃
5. **데이터 추가 폼**: 웹에서 새 매출 데이터 입력 (INSERT)
6. **CSV 다운로드**: 조회 결과를 CSV로 내려받기

## 작업 시 반드시 지킬 것

1. **SQL은 AI가 작성**한다. 수강생이 직접 SQL을 쓰지 않는다.
2. **Flask 코드도 AI가 작성**한다. 수강생은 자연어로 지시한다.
3. **법인 정렬은 가나다순**: 독일 → 미국 → 베트남 → 일본 → 중국
4. **원화 환산 = 반올림 정수**
5. **templates/ 폴더**에 HTML 파일을 둔다 (Flask 규칙)
6. **에러가 나면 AI에게 에러 메시지를 보여주고 고치게 한다**
